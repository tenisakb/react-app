### Installing

Clone the repo

```
git clone https://github.com/tenisakb/react-app.git
```

Install dependencies

```
yarn || npm i
```

Start DEVELOP

```
yarn start || npm start
```

Build PRODUCTION

```
yarn build || npm build
```

Test

```
yarn test || npm test
```
